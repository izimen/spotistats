# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.x     | :white_check_mark: |
| < 1.0   | :x:                |

## Reporting a Vulnerability

We take the security of `spotistats` seriously. If you have found a vulnerability, please **do not** open a public issue.

### Process
1.  **Email**: Send details to `security@example.com` (Replace with actual contact).
2.  **Response**: We will acknowledge receipt within 48 hours.
3.  **Resolution**: We will provide a timeline for the fix and coordinate the release.

### Scope
- **In Scope**: Source code, build dependencies, deployment configurations.
- **Out of Scope**: DDoS attacks, social engineering of team members.
